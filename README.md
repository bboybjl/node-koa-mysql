# node-koa-mysql
快速搭建koa的node后端框架数据库用mysql
# 前言
这个是单纯有注册和查看登录api的快速搭建的koa框架的封装
# 使用方法很简单
git clone https://github.com/bboybjl/node-koa-mysql  //下载项目  
npm install   //下载依赖
npm run start  //启动项目
